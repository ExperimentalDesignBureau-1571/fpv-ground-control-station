# Universal Frame

The universal frame is designed for the mounting and mechanical integration of the remote unit concentrator and peripheral devices. The remote unit concentrator, along with the peripheral devices installed on the universal frame, forms the ground station's remote unit.

The design of the frame ensures:
- installation of the remote unit concentrator
- installation of TX and VRX blocks
- unified mounting of system components
- quick transition of the remote unit between transport and working positions

<img width="600" alt="General view of the remote unit" src="picture/general_view_of_the_remote_unit.png" />

## Design Features

- The design has a modular architecture
- Provision is made for the installation of interchangeable TX and VRX blocks
- All modules are fixed with screw connections
- The design ensures resistance to mechanical loads and vibrations

## Transport Position

The transport position of the remote unit in the ground station case provides for:
- dismounted stand
- absence of a VRX block in the additional slot
- fixed position of peripheral devices
